## Description
ShoppyGlobe is a basic e-commerce application built using React and Vite.  
It demonstrates product listing, cart management, checkout flow, and routing.

## Tech Stack
- React (Vite)
- Redux Toolkit
- React Router DOM
- CSS

## Features
- Product listing from API
- Product detail page
- Add to cart / remove from cart
- Quantity management (minimum 1)
- Search using Redux state
- Checkout with order placement
- Lazy loading
- Error handling & 404 page
